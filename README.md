# Sonos Conductor

A Home Assistant custom integration that orchestrates multi-room Sonos volume:
one master volume, presence-aware zones, acoustic loudness compensation,
duck-on-door-open, dock-aware standalone mode, group repair, and HomeKit-friendly
master media player — implemented as a testable state machine instead of a pile
of YAML automations.

> **Status:** under initial development. See open pull requests for progress.
